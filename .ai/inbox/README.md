# Task Intake

This folder is for raw task intake.

Use `.ai/inbox/task-request.md` to describe a task in any format you want. Bullets, rough notes, screenshots described in text, or incomplete thoughts are all acceptable.

When a task is picked up:

1. Read the raw request.
2. Create a normalized task file in `.ai/tasks/planned/`.
3. Preserve the raw wording inside that task file.
4. Refine the task into concrete requirements, assumptions, affected files, tests, and documentation updates.
